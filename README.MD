## 项目名称
我的网络聊天室

## 项目描述

支持群聊，单聊的网络聊天室

## 使用技术
+ Java基础
+ Socket编程
+ 多线程
+ MySQL
+ JDBC

# 项目功能
+ 注册
+ 私聊
+ 群聊
+ 退出

# 项目实现
![](design.png)
+ 服务端：采用线程池调度执行服务器与客户端业务处理逻辑
+ 客户端：采用读写线程，分别处理交互与服务器数据接收

## 测试结果[可选]
+ 平均响应时间：100ms
+ 支持同时在线用户：1000-1500
